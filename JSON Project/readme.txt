Hello World!


This Python mini-project performs cleaning and analysis on a multi-nested JSON file "World Bank Project".  It uses python pandas to load, clean, and analyze.  Once loaded the columns are renamed for table visualizations.  Groupby, count, and sort is used to gather the data requested.  Blanks are filled by mapping a deep-copy of the dictionary.


Analysis performed:
* 10 countries with most # of projects
* 10 most frequent projects listed
