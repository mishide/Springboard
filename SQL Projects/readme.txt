SQL folder: performs a few quick queries for analysis of country club data from Springboards SQL database using MySQL:  https://sql.springboard.com/


Utilizing similar analysis in Python can be found in "Yammer Analysis".  Data and queries are located at Mode Analytics:  https://mode.com/sql-tutorial/a-drop-in-user-engagement/


The comparison of SQL to Python in "Yammer Analysis" is performed to visualize how Python can perform the same analysis without the need for SQL.
