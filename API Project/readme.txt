Welcome!


This Python mini-project pulls stock market data for analysis from Quandl.com.  The data is pulled via API calls utilizing available transformations to complete analysis.


Analysis performed:
* Highest & lowest opening prices
* Largest change in in a single day
* Largest change between any 2 days
* Average daily trading volume
* Median trading volume


There are 2 options to run the data
1. Via API call by providing your own API KEY and entering in line 2
2. Utilize json file included in this repo.  Skip the API call in line 2, run line 3
